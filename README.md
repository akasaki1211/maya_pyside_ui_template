# Maya PySide UI Templates
Simple templates by PySide widgets for Maya

||||
|---|---|---|
|[`basic_window.py`](basic_window.py)|Window that can configure menu bar, toolbar, and status bar|![basic_window.png](.docs/basic_window.png)|
|[`dockable_widget.py`](dockable_widget.py)|Widget that can be docked to Maya UI. Can be restored on next Maya launch.|![dockable_widget.png](.docs/dockable_widget.png)|
|[`custom_dialog.py`](custom_dialog.py)|Modal dialogs with static methods for easy use. Includes base class and input dialog.|![custom_dialog.png](.docs/custom_dialog.png)|

References:
